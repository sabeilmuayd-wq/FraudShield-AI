# 🛡️ FraudShield AI

**نظام كشف الاحتيال المصرفي بالذكاء الاصطناعي**

## 🎯 المشكلة
- 12.6 مليار شلن سُرقت في قضية واحدة (Equity Bank Kigali)
- 40% من المعاملات تأثرت بانقطاع الإنترنت
- عصابات منظمة تعمل عبر الحدود

## 🚀 الحل
- كشف فوري للمعاملات المشبوهة
- تحليل متعدد العوامل (المبلغ، الموقع، الجهاز، الوقت)
- تنبيهات فورية للبنك وهيئة الرقابة
- قاعدة بيانات مركزية للمحتالين

## 📋 الميزات
- كشف احتيال بالذكاء الاصطناعي
- قاعدة بيانات SQLite
- تنبيهات فورية
- تقارير وإحصائيات
- تسجيل قضايا الاحتيال

## 🛠️ التقنيات
- Python 3.11
- Streamlit
- SQLite
- Pandas
- Plotly

## 🚀 التشغيل

pip install -r requirements.txt
